# Golden Transcripts

These are expected-behavior transcripts for `/deep`. They are not factual source-of-truth reports; they are evaluation artifacts that show how an Organizer should move through contract, state, worker selection, reconciliation, verification, delivery, spend, and handoff.

Run the lightweight structural validator:

```bash
python scripts/validate_transcripts.py
```

Scenarios:

- [Quick fact](quick-fact.md): shallow single-action fact check.
- [Literature review](literature-review.md): medium research with state and academic evidence.
- [Decision-critical](decision-critical.md): deep contract with cross-family evidence and blind verification.
